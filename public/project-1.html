Live Link

Project Overview

This shopping app, built using Python Django for the backend, HTML and CSS for the frontend, and SQLite for the database, offers a comprehensive user experience with several key features. Users can sign up and log in to access personalized functionalities. They can browse products, add items to a cart, and mark items as favorites for future reference. The app includes a search function to help users find specific products quickly. The Django backend handles user authentication, product data management, and the logic for cart and favorites functionalities, while the frontend, crafted with HTML and CSS, ensures an intuitive and visually appealing user interface. SQLite serves as the lightweight database for storing user information, product details, and cart contents, making the app efficient and responsive.
